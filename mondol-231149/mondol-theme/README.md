# Mondol - Custom WordPress Theme

A fully responsive custom WordPress theme with Elementor Page Builder compatibility and API-based grid functionality.

## Features

✅ **Custom WordPress Theme** - Built from scratch with best practices
✅ **Elementor Compatibility** - Fully compatible with Elementor Page Builder (Free & Pro)
✅ **API Integration** - Fetches data from external WordPress REST API
✅ **Dynamic Grid Layout** - Responsive grid display with hover effects
✅ **Category Filtering** - Single-checkbox category filter with AJAX updates
✅ **Mobile Responsive** - Fully responsive design for all screen sizes
✅ **Clean Code** - Follows WordPress coding standards
✅ **Performance Optimized** - Efficient API fetching and caching
✅ **Error Handling** - Robust error handling and user feedback


## Installation

### Step 1: Upload Theme Files
1. Download the `mondol-theme` folder
2. Connect to your WordPress hosting via FTP/SFTP
3. Navigate to `/wp-content/themes/`
4. Upload the `mondol-theme` folder

OR use WordPress admin:
1. Go to **Appearance → Themes**
2. Click **Add New**
3. Click **Upload Theme**
4. Choose the theme folder and upload

### Step 2: Activate Theme
1. Go to **Appearance → Themes**
2. Find "Mondol Driving School" theme
3. Click **Activate**

### Step 3: Install Elementor (Optional)
1. Go to **Plugins → Add New**
2. Search for "Elementor"
3. Click **Install Now** and **Activate**

### Step 4: Configure Homepage
1. Go to **Settings → Reading**
2. Select **A static page** for your homepage
3. Create a new page called "Home" if needed
4. Set it as the Homepage
